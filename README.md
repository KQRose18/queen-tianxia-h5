# 女王天下

一款女性向真人短剧互动经营 H5。

玩家扮演登基第一年的女王，在国库空虚、边关不稳、朝堂暗涌的局面里，通过买卖资产、批阅诏令、提升国力和推进剧情，逐步解锁五位性格完全不同的男宠。

在线体验：[https://kqrose18.github.io/queen-tianxia-h5/](https://kqrose18.github.io/queen-tianxia-h5/)

## 核心爽点

赚钱不是目的，赚钱是门票。

诏令不是单纯策略，诏令是剧情钥匙。

排行榜不是只为了炫耀，排行榜是解锁更高级男宠剧情的资格。

整个体验围绕一条循环展开：

```text
赚钱 -> 解锁剧情 -> 解锁男宠 -> 约会互动 -> 推进短剧
```

## 你会玩到什么

- 短剧式开场：女王登基、御书房初遇、男宠引导新手流程。
- 市场经营：根据“今日风闻”买卖漕粮、云锦、盐引、兵甲、密诏、琉璃。
- 朝局诏令：每月选择一道政策，影响财富、权势、魅力、威望。
- 后宫解锁：沈砚、萧照、容与、谢长离、裴无咎五位角色逐步登场。
- 剧情互动：条漫式 CG、角色台词、选择分支、好感反馈。
- 声音与反馈：BGM、交易音效、真人语音切片与结算动效。

## 五位男宠

**沈砚**  
白切黑首辅。克制、温柔、占有欲强。关键词：御书房、白衣、晨光、眼镜链。

**萧照**  
疯犬将军。忠诚、危险、锋利。关键词：战场、黑甲、擦刀。

**容与**  
异域皇子。撩人、浪漫、游刃有余。关键词：宫宴、葡萄酒、丝路。

**谢长离**  
奸商。腹黑、会赚钱、会算人心。关键词：账房、算盘、铜钱。

**裴无咎**  
杀手。疯批、冷冽、偏执守护。关键词：夜雨、屋顶、刀锋。

## 当前版本

这是一个可外发体验的 MVP 版本，重点验证：

- 女性向真人短剧 H5 的第一分钟吸引力。
- “经营赚钱 -> 剧情推进 -> 男宠解锁”的留存循环。
- 条漫 CG、语音、BGM、市场与朝局系统能否组成完整可玩闭环。

部分剧情图和声音仍处于可发布占位或阶段性补图状态，后续会继续替换为正式美术与更完整的逐句角色语音。

## 技术说明

项目是纯静态 H5，可部署到 GitHub Pages、Cloudflare Pages、Vercel、Netlify 等静态托管平台。

发布包会自动压缩图片与音频：

- 图片转 WebP。
- BGM / 音效 / 语音压缩为 MP3。
- 发布前校验静态资源引用和语音清单，避免缺图、缺音频。
